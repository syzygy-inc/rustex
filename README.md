# rustex → SabiTeX

This project has been transferred to **SabiTeX** (<https://github.com/syzygy-inc/SabiTeX>).

The purpose is to retire the name "RusTeX", out of respect for the pre-existing project of the same name: <https://github.com/FlexiFormal/RusTeX>.

This repository will be kept as an archive, in order to keep the project's history clear.
